## 🚀 核心看点：一键安装与快捷启动

别废话，直接上车！在你的 Linux 终端（支持 Ubuntu/Debian/CentOS 等，需 `root` 权限）中直接复制并运行以下命令：

```bash
sudo curl -sSL [https://raw.githubusercontent.com/qmsdh/qmstool/main/qmstool.sh](https://raw.githubusercontent.com/qmsdh/qmstool/main/qmstool.sh) -o /usr/local/bin/qmstool && sudo chmod +x /usr/local/bin/qmstool && qmstool
```

**💡 为什么这行命令最优雅？**
1. **系统级映射**：它会自动将工具箱下载至 `/usr/local/bin/` 并隐去 `.sh` 后缀，使其伪装成系统原生命令。
2. **永久快捷键**：首次运行后，以后在任何路径、任何目录下，**只需在终端输入 `qmstool` 即可秒开工具箱**，再也不用背又长又臭的下载命令了！

## 🛠 功能全景：五大核心神器聚合

工具箱目前精心挑选并重构了 5 大高频硬核功能，分为【自研核心】与【网络集成】两大板块：

### 1️⃣ 一键修改高级 DNS (`dns_set.sh`)
* **痛点**：很多海外 VPS 默认的 DNS 解析极慢，甚至经常导致拉取 GitHub 失败。
* **特性**：内置 Cloudflare (1.1.1.1)、Google (8.8.8.8)、阿里云、腾讯云等优质解析源，同时支持**自定义 DNS 输入**。一键覆盖 `/etc/resolv.conf`，让网络瞬间起飞。

### 2️⃣ Linux Swap 交换分区管理 (`swap_set.sh`)
* **痛点**：买的高性价比小鸡内存太小，稍微跑点吃内存的程序就直接 OOM 崩溃？
* **特性**：支持一键动态创建、挂载或调整系统 Swap 虚拟内存。提供**【临时生效】**与**【永久生效（写入fstab）】**双模式，也支持一键彻底关闭并清理旧 Swap 空间。

### 3️⃣ Alist 网盘多开管理增强版 (`alist_set.sh`)
* **痛点**：一个 Alist 挂载不够用，想在一台机器上开多个独立的 Alist 实例？
* **特性**：支持**多实例无限多开**！集成一键安装新实例、更新指定实例、重置密码、独立管理（启动/停止/重启）以及查看所有已安装实例的端口和目录状态。

### 4️⃣ 一键搭建 Vless 节点 (Sing-Box 内核)
* **特性**：直接动态调用网络上最顶尖的 Sing-Box 一键搭建脚本。全自动配置通用网络环境，一步到位，带你畅游网络。

### 5️⃣ 官方原版 Speedtest 测速
* **特性**：一键自动配置系统的 Packagecloud 官方源，静默安装 Ookla 官方原生 Speedtest 命令行工具并直接开跑。拒绝山寨测速，数据精准靠谱。

## 🛡 纯内存执行：这很 Geek！

传统工具箱往往会把你整套项目的脚本全部下载到用户的服务器上，占用空间不说，还容易留下各种配置垃圾。

**QMS 工具箱采用了更高级的“纯内存解析运行（Zero-Footprint Execution）”机制**：
除了主控程序 `qmstool` 映射在系统里，当你选择修改 DNS 或配置 Swap 时，主程序会通过管道流 `bash <(curl -sSL ...)` 直接从 GitHub 抓取子脚本并丢进**内存**中解析执行。
* 运行完毕后，**本地不留存任何子脚本文件**，随用随走，极致干净！
* **永远是最最新版**：由于子脚本每次都在云端抓取，只要作者在 GitHub 仓库提交了优化，你无需更新工具箱，直接点选该功能就是最新版！

## 🔄 完善的工具箱自更新机制

如果主菜单增加了新功能，或者主控程序修复了 Bug 怎么办？

工具箱内置了 **【6. 检查并更新主控程序】** 功能。它会动态比对本地与 GitHub 仓库根目录下的 `version.txt` 文件。一旦发现云端有版本迭代，会提示你进行一键自动下载并无缝覆盖本地的 `/usr/local/bin/qmstool`，实现真正的无痛升级。
